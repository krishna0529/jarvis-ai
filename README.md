An Industry-Level AI Assistant


Features

- Voice Assistant
- AI Brain
- Automation
- Vision
- Memory
- Plugins
- Offline Support



✅ Installation
First, clone the repository:

git clone https://github.com/zoro-san/JARVIS-AI-0.2.git
cd JARVIS-AI-0.2

✅ Install Dependencies:

pip install -r requirements.txt

✅ Configuration:

Create a .env file:

OPENAI_API_KEY=sk-proj-...

✅ Run:

python main.py
