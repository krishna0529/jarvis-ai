# JARVIS Enterprise Memory System

Modular memory subsystem containing:
- **Short-Term Memory**: In-memory session store.
- **Long-Term Memory**: Persistent SQLite database.
- **Episodic Memory**: Experiences tracking.
- **Semantic Memory**: Embeddings-based similarity search using Sentence Transformers and FAISS.
- **Memory Consolidation**: Scoring logic to promote short-term data to long-term database.
